## 💡 **About Gameox**

**Gameox** – *Gesture Operated Game (Personal R&D)*

Gameox represents a breakthrough in **human-computer interaction**, merging AI vision systems with intuitive gaming mechanics. This gesture-controlled space shooter replaces traditional keyboard inputs with **real-time hand gesture recognition**, allowing players to control the game through natural body movements.

### 🎯 **Project Vision**
> *Building intuitive, AI-driven systems that redefine how humans interact with technology through touchless interfaces and natural human movement.*

---

## ✨ **Key Features**

✅ **Real-Time Hand Gesture Recognition** – Camera-based input system using Computer Vision  
✅ **Touchless Gaming Interface** – No keyboard/mouse required—pure gesture control  
✅ **AI Vision Pipeline** – Intelligent hand tracking and motion detection  
✅ **Interactive Space Shooter Gameplay** – Engaging game mechanics powered by gestures  
✅ **End-to-End Development** – Solo project covering core logic, vision pipeline, and UI  
✅ **Web-Based Deployment** – Play directly in your browser

## 🎮 **How to Play**

### **Controls**
- 👋 **Hand Position** – Move your hand across the camera frame to control the spaceship
- ✋ **Open Palm** – Shoot projectiles at enemies
- ✌️ **Victory Sign** – Special moves (varies by game mode)
- 🙌 **Both Hands** – Shield activation / Power-up trigger

### **Gameplay Tips**
1. Ensure good lighting for optimal hand detection
2. Keep your hand visible to the camera at all times
3. Use smooth, deliberate gestures for better accuracy
4. Focus on hitting enemies while avoiding obstacles
5. Collect power-ups to enhance your spaceship capabilities

## 🚀 **Installation & Setup**

### **Requirements**
- Python 3.x
- OpenCV (`opencv-python`)
- Webcam/Camera device
- Modern web browser (Chrome, Firefox, Safari, Edge)

### **Quick Start**

1. **Clone the repository**
   ```bash
   git clone https://github.com/SONIC445-BYTE/Test-gameox.git
   cd Test-gameox
   ```

2. **Install Python dependencies**
   ```bash
   pip install opencv-python
   ```

3. **Run the application**
   ```bash
   python main.py
   ```

4. **Access the game**
   - Open `index_v0.2.html` in your web browser
   - Allow camera access when prompted
   - Start playing!

## 🔬 **Technical Deep Dive**

### **Architecture Overview**
```
Camera Input 
    ↓
OpenCV Gesture Recognition Pipeline
    ↓
Hand Detection & Tracking
    ↓
Gesture Classification
    ↓
Game State Update
    ↓
HTML5 Canvas Rendering
    ↓
Player Feedback
```

### **Key Innovations**
- **Real-time Processing**: Sub-100ms gesture response time
- **Robust Detection**: Handles varying lighting conditions and hand sizes
- **Seamless Integration**: Vision system integrated directly with game loop
- **Scalable Architecture**: Modular design for future gesture additions
